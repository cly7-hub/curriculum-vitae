# 图片资源文件夹

## 个人照片

请将您的个人照片放在此文件夹中，命名为：

- `photo.jpg` - 个人头像照片（用于首页展示）

**照片要求**：
- 格式：JPG、PNG 或 WebP
- 建议尺寸：400x400 像素或更高（正方形）
- 文件大小：建议小于 300KB
- 建议：正面照片，背景简洁，适合圆形裁剪

## UG设计作品图片

请将您的UG设计案例图片放在此文件夹中，命名为：

- `ug-case1.jpg` - 扭曲圆柱体设计
- `ug-case2.jpg` - 机械连接件设计  
- `ug-case3.jpg` - U型夹设计

**图片要求**：
- 格式：JPG、PNG 或 WebP
- 建议尺寸：800x600 像素或更高
- 文件大小：建议小于 500KB（可压缩优化）

## 使用说明

- 个人照片会自动显示在首页hero区域
- UG设计案例图片会自动显示在"UG软件建模项目"卡片中，垂直排列展示
